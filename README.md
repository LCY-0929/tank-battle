# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 和 JavaScript 制作的经典坦克大战游戏。

## 🎯 游戏特色

- **经典玩法**: 控制坦克消灭所有敌人，通过关卡
- **AI 敌人**: 智能 AI 敌人会追踪、射击和躲避
- **多种道具**: 速度提升、快速射击、护盾、多重射击、生命恢复
- **可破坏地形**: 砖墙可被子弹摧毁，钢铁墙不可摧毁
- **关卡系统**: 三种不同的地图布局循环，难度逐级递增
- **小地图**: 右上角小地图显示战场态势
- **粒子特效**: 爆炸、弹道拖尾等视觉效果
- **屏幕震动**: 击杀和受伤时的屏幕震动反馈
- **触屏支持**: 支持移动端触屏操作

## 🕹️ 操作方式

| 按键 | 操作 |
|------|------|
| `W A S D` | 移动坦克 |
| `鼠标移动` | 瞄准 |
| `鼠标左键` | 开火 |
| `R` | 快速装弹 |
| `空格` | 暂停/继续 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

```bash
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📝 游戏规则

1. 消灭当前关卡所有敌方坦克即可进入下一关
2. 玩家有 3 条生命，被击中会损失一条命
3. 每 3 关奖励一条额外生命
4. 收集地图上随机出现的道具获得增益效果
5. 道具效果持续 10 秒
